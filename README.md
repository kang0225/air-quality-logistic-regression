# 🏙️ 서울시 초미세먼지 '나쁨' 등급 예측 프로젝트

> 서울시 공공데이터를 활용하여 내일의 초미세먼지(PM2.5) 등급이 '나쁨' 이상일지를 예측하는 로지스틱 회귀 모델을 개발합니다. 이 프로젝트는 데이터 수집, 전처리, 모델링, 평가의 전 과정을 포함하며, 시민 건강과 직결되는 환경 문제 해결에 기여하는 것을 목표로 합니다.

---

## 💡 프로젝트 개요

* **프로젝트 목표**: 서울시의 대기질 및 기상 데이터를 기반으로 다음 날의 초미세먼지(PM2.5) 등급이 '나쁨(36µg/m³ 이상)' 또는 '매우나쁨'에 해당하는지를 예측하는 이진 분류 모델 구축
* **주요 모델**: 로지스틱 회귀 (Logistic Regression)
* **프로젝트 기간**: * **데이터 출처**:
    1.  [서울시 일별 평균 대기오염도 정보](https://data.seoul.go.kr/dataList/OA-21228/S/1/datasetView.do)
    2.  [기상자료개방포털 (ASOS 일별 자료)](https://data.kma.go.kr/data/grnd/selectAsosRltmList.do?pgmNo=36)

## 📊 프로젝트 진행 계획

### 1. 문제 정의 (Problem Definition)
- [x] 예측 목표 설정: 내일의 초미세먼지 등급이 '나쁨' 이상인지 여부 (Binary Classification)
- [x] 모델 선정: 로지스틱 회귀 모델을 기본 모델로 사용
- [ ] 평가지표 선정: Accuracy, Precision, Recall, F1-Score, ROC-AUC Score

### 2. 데이터 수집 (Data Collection)
- [x] 서울시 일별 평균 대기오염도 데이터 다운로드
- [x] 기상자료개방포털에서 서울 지역 일별 기상 데이터 다운로드

### 3. 데이터 전처리 및 EDA (Exploratory Data Analysis)
- [ ] 두 데이터셋(`대기질`, `기상`)을 '날짜' 기준으로 병합 (`merge`)
- [ ] 데이터 기본 정보 확인 (결측치, 데이터 타입 등)
- [ ] 결측치 처리 방안 수립 및 실행
- [ ] 목표 변수(Target) 생성: PM2.5 농도를 기준으로 '나쁨' 이상(1), '보통' 이하(0)로 변환
- [ ] 데이터 시각화를 통한 변수 간 관계 및 분포 탐색 (EDA)

### 4. 특성 공학 (Feature Engineering)
- [ ] 어제, 그저께 농도 등 시차(Lag) 특성 생성
- [ ] 이동 평균(Moving Average) 특성 생성
- [ ] 요일, 월 등 날짜 관련 파생 변수 생성
- [ ] 최종 모델링에 사용할 특성 선택

### 5. 모델링 (Modeling)
- [ ] 데이터를 훈련(Train) 세트와 테스트(Test) 세트로 분리
- [ ] 데이터 스케일링 (StandardScaler)
- [ ] 로지스틱 회귀 모델 훈련
- [ ] (선택) 다른 분류 모델과 성능 비교 (예: 의사결정나무, 랜덤 포레스트)

### 6. 모델 평가 (Evaluation)
- [ ] 테스트 세트로 모델 성능 평가
- [ ] 혼동 행렬(Confusion Matrix) 시각화
- [ ] 주요 평가지표(Accuracy, F1-Score 등) 계산 및 분석
- [ ] ROC Curve 시각화 및 AUC 점수 확인

### 7. 결과 해석 및 보고
- [ ] 프로젝트 결과 보고서 작성
- [ ] 모델의 회귀 계수를 통해 어떤 변수가 예측에 중요한 영향을 미치는지 분석
